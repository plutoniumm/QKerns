Kernel Functions

The ultimate objective of a kernel is to use some fancy way to still tell us how similar any two inputs $x$, $x'$ are and map it to a higher dimensions. Some commonly used kernels are as follows. Our objective here is to define some basic kernels and then see how we can use them to combine them and generate fancier functions

Baseline testing with just to see things work

Here we plot the function $F(x, y) = x * y$ simply for an established baseline.

RBF Kernel: $F(x, y) = e^{-\gamma * (x - y)^2}$

Here we plot the function $F(x, y) = e^{-\gamma * (x - y)^2}$, where $gamma$ is a hyperparameter.

Sigmoid Kernel: $F(x, y) = \tanh(\gamma * (x - y) + \beta)$

Here we plot the function $F(x, y) = \tanh(\gamma * (x - y) + \beta)$, where $\gamma$ and $\beta=0.5$ are hyperparameters.

Polynomial Kernel: $F(x, y) = (\gamma * (x - y) + \beta)^d$

Here we plot the function $F(x, y) = (\gamma * (x - y) + \beta)^d$, where $\gamma$, $\beta=0.5$ and $d=2$ are hyperparameters.

Matern Kernel: $F(x, y) = \frac{2^{1-\nu}}{\Gamma(\nu)}\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)^\nu K_\nu\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)$

Here we plot the function $F(x, y) = \frac{2^{1-\nu}}{\Gamma(\nu)}\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)^\nu K_\nu\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)$, where $\nu=1.5$ and $\lambda=1$ are hyperparameters.

Periodic Kernel: $F(x, y) = \exp\left(-\frac{2\sin^2\left(\pi\frac{|x - y|}{p}\right)}{l^2}\right)$

Here we plot the function $F(x, y) = \exp\left(-\frac{2\sin^2\left(\pi\frac{|x - y|}{p}\right)}{l^2}\right)$, where $l=1$ and $p=1$ are hyperparameters.

RQ Kernel: $F(x, y) = \left(1 + \frac{|x - y|^2}{2\alpha l^2}\right)^{-\alpha}$

Here we plot the function $F(x, y) = \left(1 + \frac{|x - y|^2}{2\alpha l^2}\right)^{-\alpha}$, where $\alpha=1$ and $l=1$ are hyperparameters.

White Noise Kernel: $F(x, y) = v\delta(x - y)$

Here we plot the function $F(x, y) = v\delta(x - y)$, where $v=1$ is a hyperparameter.
