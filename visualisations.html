Kernel Functions

Baseline testing with just to see things work

Here we plot the function $F(x, y) = x * y$ simply for an established baseline.

RBF Kernel: $F(x, y) = e^{-\gamma * (x - y)^2}$

Here we plot the function $F(x, y) = e^{-\gamma * (x - y)^2}$, where $gamma$ is a hyperparameter.

Laplacian Kernel: $F(x, y) = e^{-\gamma * |x - y|}$

Here we plot the function $F(x, y) = e^{-\gamma * |x - y|}$, where $gamma$ is a hyperparameter.

Sigmoid Kernel: $F(x, y) = \tanh(\gamma * (x - y) + \beta)$

Here we plot the function $F(x, y) = \tanh(\gamma * (x - y) + \beta)$, where $\gamma$ and $\beta=0.5$ are hyperparameters.

Polynomial Kernel: $F(x, y) = (\gamma * (x - y) + \beta)^d$

Here we plot the function $F(x, y) = (\gamma * (x - y) + \beta)^d$, where $\gamma$, $\beta=0.5$ and $d=2$ are hyperparameters.

Matern Kernel: $F(x, y) = \frac{2^{1-\nu}}{\Gamma(\nu)}\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)^\nu K_\nu\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)$

Here we plot the function $F(x, y) = \frac{2^{1-\nu}}{\Gamma(\nu)}\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)^\nu K_\nu\left(\sqrt{2\nu}\frac{|x - y|}{\lambda}\right)$, where $\nu=1.5$ and $\lambda=1$ are hyperparameters.
